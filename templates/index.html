RepoRef: Repository reference

This website allows you to reference to a certain commit or branch in a public git repository, by URL. For instance: `git clone http://reporef.org/github.com/GeertJohan/yubigo@37f797f35ecc54a3e55d3816c73d2f28fb763e01/.git/` clones the yubigo repository directly at commit 37f797f[...].

Go get

"go get" is a command in the go-tool for the Go programming language. It is used to retrieve and install a library (or: package) from a public git reporitory. RepoRef.org can be used to easily go get a package at a certain commit or branch. Just execute go get reporef.org/github.com/user/project@branch-name

GitHub

At this point RepoRef only supports GitHub. Support for other public repositories is scheduled. Please join and help.
